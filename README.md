# 🏧 Sistema Caixa Eletrônico

Sistema simulando operações de um caixa eletrônico, desenvolvido em Java com foco em Orientação a Objetos.

## 💻 Tecnologias
- Java 25
- IntelliJ IDEA Community

## ⚙️ Funcionalidades
- Cadastro de conta com titular, login e senha
- Verificação de senha com bloqueio após 3 tentativas
- Saque com validações de saldo, valor mínimo e conta bloqueada
- Login com verificação de usuário
- Menu interativo com Scanner
- Consulta de saldo
- Depósito

## 🚧 Em desenvolvimento

- Histórico de transações
